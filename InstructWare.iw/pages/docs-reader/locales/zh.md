# Docs Reader Copy (zh)

## Header

- header.current_doc_title: 当前文档
- header.current_doc_status: 状态
- header.current_locale: 语言

## Toolbar

- toolbar.switch_manifesto: 切换到 Manifesto
- toolbar.switch_protocol: 切换到 Protocol
- toolbar.back_docs_center: 返回文档中心

## Viewport

- viewport.loading: 正在加载 Markdown 内容...
- viewport.empty: 当前文档内容不可用。

## Footer

- footer.progress_hint: 阅读进度会同步到运行时状态。
- footer.open_source_markdown: 打开源 Markdown
