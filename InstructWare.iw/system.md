# System Context: InstructWare.org Official Site

## Product Identity

InstructWare.org is the official site for the InstructWare protocol, manifesto, and page-first intent authoring workflow.

## Global Rules

- Keep intent markdown human-readable and page-first.
- Keep runtime behavior traceable from source intent to implementation links.
- Keep protocol statements verifiable and avoid speculative autonomy claims.

## Information Architecture

- Home page explains core thesis and onboarding path.
- Docs Center routes users to Manifesto and Protocol reading flows.
- Docs Reader renders long-form markdown with deterministic navigation and progress state.

## Runtime Guarantees

- Intent and implementation changes remain reviewable, testable, and reversible.
- Source intent remains authoritative over generated or inferred runtime behavior.

## Security Model

- External links open with secure browser policy and no unsafe window access.
- Runtime changes require explicit validation gates before delivery.
